# Structurizr

Structurizr is a "models as code" tool designed for the [C4 model](https://c4model.com) - you write [Structurizr DSL](/dsl) and Structurizr renders diagrams:

- [Structurizr DSL - Example](/dsl/example)
- [Structurizr DSL - Tutorial](/dsl/tutorial)

> Structurizr is not a traditional "drag and drop" tool and does not provide a way to make architecture diagrams via a UI. See [UI-driven vs "as code"](https://www.patreon.com/posts/151073821) for the rationale behind this.

## Summary

- Create software architecture models using the [Structurizr DSL](/dsl/example), [Java](/java/getting-started), and [more](/community#authoring-tools).
- Create multiple diagrams from a single architecture model ([tutorial](/dsl/tutorial)).
- Manual and automatic layout are supported via the browser-based [diagram editor](/server/diagrams/editor).
- Rendering tool independent - [Structurizr diagram viewer](/server/diagrams/viewer), [Structurizr graph viewer](/server/additional-visualisations), [PlantUML](/getting-started/export-diagrams-as-code#plantuml-examples), [Mermaid](/getting-started/export-diagrams-as-code#mermaid-example), and [more](/community#rendering-tools).
- [Open source](https://github.com/structurizr/structurizr)

Visit [playground.structurizr.com](https://playground.structurizr.com) to try Structurizr without installing any tooling.
