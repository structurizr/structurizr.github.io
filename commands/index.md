# Commands

All the Structurizr commands listed below are free and open source except for [server](/server) - this follows an open core model, with closed extensions for features that are designed for use in enterprise environments.


| Command                   | Description                                                                                                                   |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [playground](/playground) | Starts the Structurizr playground.                                                                                            |
| [local](/local)           | Starts Structurizr local, which provides a way to view diagrams and modify their layout (the equivalent of Structurizr Lite). |
| [server](/server)         | Starts a Structurizr server (the equivalent of the Structurizr on-premises installation).                                     |
| [export](/export)         | Exports the views in a Structurizr workspace to PlantUML, Mermaid, or a static HTML site.                                     |
| [push](/push)             | Pushes a workspace to a Structurizr server.                                                                                   |
| [pull](/pull)             | Pulls a workspace from a Structurizr server.                                                                                  |
| [branches](/branches)     | Lists the branches for a workspace.                                                                                           |
| [delete](/delete)         | Deletes a workspace branch.                                                                                                   |
| [lock](/lock)             | Locks a workspace on a Structurizr server.                                                                                    |
| [unlock](/unlock)         | Unlocks a workspace on a Structurizr server.                                                                                  |
| [merge](/merge)           | Merges the layout information into a workspace.                                                                               |
| [validate](/validate)     | Validates a workspace.                                                                                                        |
| [inspect](/inspect)       | Runs the inspections on a workspace.                                                                                          |
| [list](/list)             | Lists the elements in a workspace.                                                                                            |
| version                   | Displays version information.                                                                                                 |
| help                      | Displays a help message.                                                                                                      |
